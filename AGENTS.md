# Bukyung Wiki Public-Lite Rules

This is the public, PDF-free demo package of Bukyung Wiki.

## What Is Included

- `index.html` static frontend
- `_data.json` public metadata and graph edges
- `index.md` paper list
- `wiki/` public wiki notes and overview documents

## What Is Not Included

- no `papers/` PDF files
- no `sources/` LLM source summaries
- no raw extracted text
- no private batch reports or working logs

## Answering Rule

Use only the files included in this public package. If a requested paper,
PDF, source summary, or raw extraction is not present here, say that it is
not included in this public demo package. Do not infer missing paper details
from general knowledge or web search.

For full private rebuilds, create a separate private working copy and add
local PDFs under `papers/`; do not commit or redistribute those PDFs.
